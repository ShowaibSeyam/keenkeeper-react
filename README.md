Name of the Peoject : Keenkeeper-react

Description: My project is used to keep friend close. Cause friend is the most valuable wealth we can make in this world. 
So my app is doing research to all of my friend like who is getting time from me and who is neglected. so it works on
keeping the relationship strong and healthy.

Technology have been used:
1. React.js for Build the UI
2. React Router DOM to Handle page navigation
3. Tailwind CSS for	Styling and responsiveness
4. Used Recharts for showing the analytics page

Feature: 
  1. Showing every friends information
  2. Timeline shows the occurences
  3. Stats shows friendship is easy to read and maintain
